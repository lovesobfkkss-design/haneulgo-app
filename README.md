# 하늘고 학생 자료 배부 체크리스트

학생들의 자료 배부 현황을 주차별로 관리하는 웹 애플리케이션입니다.

## 주요 기능

- 📅 주차별 배부 현황 관리
- ✅ 학생별 체크리스트
- 👥 학생 추가/삭제
- 📚 과목 편집
- 🔍 검색 및 필터링
- 💾 자동 저장 (localStorage)

## 사용 방법

웹 브라우저에서 `index.html` 파일을 열거나 호스팅된 URL로 접속하세요.

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage (데이터 저장)

## 웹에서 접근하는 방법

이 애플리케이션을 인터넷에서 누구나 접근할 수 있도록 호스팅하려면 아래 방법 중 하나를 선택하세요.

### 방법 1: Netlify Drop (가장 쉬움) ⭐ 추천

1. [Netlify Drop](https://app.netlify.com/drop) 사이트로 이동
2. `index.html` 파일을 드래그 앤 드롭
3. 즉시 URL 생성! (예: `https://random-name.netlify.app`)
4. 로그인하면 도메인 이름 변경 가능

**장점**: 가입 불필요, 30초 안에 완료

### 방법 2: GitHub Pages (영구적)

1. GitHub 계정 생성 (없는 경우)
2. 새 레포지토리 생성
3. 파일 업로드
4. Settings > Pages에서 활성화
5. URL: `https://사용자명.github.io/레포지토리명`

**장점**: 무료, 영구적, 버전 관리

### 방법 3: Vercel (빠르고 쉬움)

1. [Vercel](https://vercel.com) 가입
2. "New Project" 클릭
3. 파일 업로드 또는 GitHub 연동
4. 자동 배포
5. URL: `https://프로젝트명.vercel.app`

**장점**: 빠른 배포, 자동 HTTPS
